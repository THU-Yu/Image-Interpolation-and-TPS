# Image Interpolation and TPS
Platform: Window10
Version: 1.0.0
Author: Chen YuHong

## Introduction
本项目制作了用户交互界面，提供用户从特定路径读取图片或保存图片至指定路径，
详细的界面操作指南请见报告。

## Environment
程序运行环境：VS2019 Community, .Net Framework版本为4.7.2, 使用库函数皆为标准界面库函数，无需安装特定的库函数。
